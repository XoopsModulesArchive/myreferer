Informations about myReferer


This module will gather all informations regarding your referers from external websites and search engines. It will help you track down used keywords and web crawlers.

For more informations, read the following explanations
  • 1) Installation
  • 2) Referers
  • 3) Search Engine
  • 4) Keywords
  • 5) Web Crawlers

1) Installation

 Follow the next steps to properly install the module:
  1. 1) Edit the 'header.php' file at the Xoops root directory.
  2. 2) Insert the code as indicated below : 
    if (!defined("XOOPS_ROOT_PATH")) {
    die("XOOPS root path not defined");
}
include(XOOPS_ROOT_PATH.'/modules/myreferer/record.php');
  3. 3) Reupload the modified file.

That's it.
Pay attention, if your update you Xoops site, and that the 'header.php', the module won't work anymore! Make the above operation again.

2) Referers
A misspelling of 'referrer' which somehow made it into the {HTTP} standard. A given {web page}'s referer (sic) is the {URL} of whatever web page contains the link that the user followed to the current page. Most browsers pass this information as part of a request.

3) Search Engine
A program that searches documents for specified keywords and returns a list of the documents where the keywords were found. Although search engine is really a general class of programs, the term is often used to specifically describe systems like Google, Alta Vista or Excite that enable users to search for documents on the World Wide Web.

4) Keywords
Keyword is the term used for words included in a web page that would match words used by web surfers in finding that web page (via a search engine). Keywords can simply be words included in the body text of the document, or added to the header using meta tags to increase the number of keywords. Selecting keywords, that match your target audiences' use of the web is a critical marketing tactic.

5) Web Crawlers
A Crawler (which may also be called a robot, spider, or bot) is a program that automatically traverses the Web's hypertext structure by retrieving a document, and recursively retrieving all documents that are referenced. For more information on Crawlers and the standards of crawling which we follow, you can visit the WebRobots FAQ (http://www.robotstxt.org/wc/robots.html).
